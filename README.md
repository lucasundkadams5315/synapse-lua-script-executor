## About Synapse

Synapse is a Windows desktop application for Roblox Lua injection, script execution, and routine automation tasks. Its compact layout and direct startup process are designed to help users begin scripting quickly without a complicated setup.

Alongside its executor functions, the application provides a 500+ item script hub, saved queue support, sequential batch execution, and an integrated update engine. Synapse also offers multiple interface languages and an offline local UI for users who prefer working from a Windows desktop application instead of through a browser.
